# Rhymes

A rhyming word compilation of all my verses.

## Format

- `{}` 所改编的原诗的未改动的韵脚
- `[]` 非连续出现的重复韵脚
- `<>` 连续出现的重复韵脚\
  以上 3 类仅首尾字计作有效韵脚。
- `//` 他人作品，但 unt 认可其韵感并传唱，正常计入（部分不应计入的他人作品已排除）
- `；` 换韵
- `‖` 交韵（即 ABAB 式押韵）
